# awesome-reader

一些阅读者的好用工具，打造阅读与输出的舒适流程。因地域使用局限，暂只涉及中文用户。

本仓库条目内容均在 [zotero-awesome_reader](https://www.zotero.org/groups/4993351/awesome-reader/) 按对应目录储存

发起这个项目并搭建框架时，发现这个内容量并不是我现在可以承载的。遂决定定期更新内容。

## 结构说明

本篇按阅读流程四步走

- 图书获取
- 图书管理
- 图书阅读
- 阅读输出

在每个流程中分别介绍软件、实用好物
相关书单、书籍可能不止涉猎一个流程，书单单独放置一个分类。
